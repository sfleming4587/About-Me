Kitchen Chaos

Welcome to Kitchen Chaos, a fast-paced cooking simulation game developed using Unity and hosted via WebGL. This project is a demonstration of my ability to quickly learn and apply new skills in software development, based on a detailed tutorial by Code Monkey on YouTube.

What I Learned

This project, Kitchen Chaos, is not an original work but rather an implementation of a comprehensive tutorial by Code Monkey, which I followed to enhance my skills as part of my software development job search. Through this process, I gained hands-on experience with the Unity game engine and C# programming language, two powerful tools in modern game development. I learned to leverage Unity’s robust features, such as its scene management and component-based architecture, to build an interactive game environment. A key takeaway was mastering animations within Unity—using Animator components and state machines to bring characters and objects to life, ensuring smooth transitions and responsive gameplay. Additionally, I explored C# delegates, a concept that allowed me to implement flexible event-handling systems, making the game logic more modular and efficient.
